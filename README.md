## Overview

Expense Tracker Dashboard is a full-stack MERN application that helps users manage their income and expenses efficiently. The application provides secure authentication, transaction management, financial analytics, and visual reports through an interactive dashboard.

## Features

* User Registration and Login
* JWT Authentication
* Protected Routes
* Add Transactions
* Delete Transactions
* Dashboard Analytics
* Income and Expense Tracking
* Balance Calculation
* Pie Chart Visualization
* Responsive User Interface

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* Tailwind CSS
* Recharts

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* BcryptJS

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the backend folder:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
